SCMapDB Repacker

A script on this server downloads and repacks every map on SCMapDB to fix various problems, such as:
  • missing files
  • unused files
  • broken .res files
  • files that override default content
  • inconsistent folder layouts
  • incorrectly placed/named files
The bulk of the work is done by resguy. The script runs every hour to check for updates.

Repack results

A filterable list of all maps that were repacked by the tool, each with a detailed view of changes. This doubles as a working search feature for SCMapDB, which hasn't had a working search bar for several years.

Warning: maps with scripts in them may be repacked incorrectly. It's sometimes impossible to know which files a script precaches, so the repacker likely deletes more than it should.

Content pool

A filterable list of every file that custom maps include. Files in this pool are used to recover missing files in map archives.

Each file entry lists the map(s) that reference it. Files highlighted in red are conflicted and require a manual repack to fix.
